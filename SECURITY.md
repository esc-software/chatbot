# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to our official security portal:

**https://security.esc-software.com**

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Responsible Disclosure

We kindly ask that you:

- Do not publicly disclose the vulnerability before we have had a chance to address it.
- Provide sufficient details to reproduce the issue.
- Allow reasonable time for a fix to be deployed.

Thank you for helping keep this project safe.
